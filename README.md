# tweepy

Useful script to collect tweets through tweepy API.

Tasks:
- Setting up Keys and token
- Creating an API object
- Establishing criteria for the query and its filters
- Setting up cursor
- Creating a list of tweets
- Pre-processing
  + Mention
  + Reserved
  + URL
  + Numbers
  + Hashtag
  + Lowercase
  + French stopwords
  + punctuation
  + Tokenizing
- Saving data to file
